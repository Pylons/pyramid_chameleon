Chameleon templating system Bindings for Pyramid
=================================================

These are bindings for the `Chameleon templating system
<https://chameleon.readthedocs.io/en/latest/>`_ for the Pyramid_ web framework.  See
https://docs.pylonsproject.org/projects/pyramid_chameleon/en/latest/ for
documentation.

.. _Pyramid: https://trypyramid.com/
